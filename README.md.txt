# Hope For Strays

Simple static fundraising website for dog rescue and adoption.

## Files
- index.html
- style.css
- script.js

## Deploy
Upload to GitHub and enable GitHub Pages from:
Settings → Pages → Deploy from branch → main → /(root)

## Edit later
Replace:
- UPI ID
- UPI Name
- WhatsApp number if needed
- Internet photos with real rescue photos
- Rescue stories with actual cases
